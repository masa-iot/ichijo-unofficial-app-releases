# ICHIJO施主手帳【非公式】

一条工務店の施主のための図面・見積り管理アプリ

**[アプリ紹介ページ](https://masa-iot.github.io/ichijo-unofficial-app-releases/)**

## ダウンロード

[Releases](https://github.com/masa-iot/ichijo-unofficial-app-releases/releases) からAPKをダウンロードしてください。

## 主な機能

- **ドキュメント管理** — 図面番号ベースで全書類を一元管理
- **図面チェック** — CAD・電気図面の2バージョン差分比較
- **見積り管理** — 自動パース・2版比較・推移表・CSVエクスポート
- **ウチメモ+** — 図面や見積りにその場でメモ、元の画面にワンタップで戻れる

## 注意事項

このアプリは一条工務店の公式アプリではありません。施主個人が開発した非公式アプリです。
