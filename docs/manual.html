はじめに

このアプリについて

ICHIJO施主手帳（非公式）は、一条工務店の施主が自分の打ち合わせを効率化するために作った非公式アプリです。一条公式「iサポ」で届くPDFを取り込むだけで、以下のことが片手でできます。

  • ドキュメント管理：図面番号ベースでグループ化
  • 図面チェック：CAD・電気図面のピクセル差分を色分け表示
  • 見積り管理：2版比較・推移表
  • 仕様書チェック：設備仕様書のセクション別差分
  • ウチメモ+：画面上の位置にピン留めしてメモ
  • ブックマーク：よく見る画面にワンタップアクセス
  • スケジュール：家づくり工程のタイムライン＆カウントダウン
  • ホームウィジェット：端末ホーム画面にカウントダウン表示
  • でんき通信簿（入居後）：太陽光・蓄電池の経済効果を可視化
  • 便利ツール：湿度計算、外部リンク集など

🔒 すべてのデータは端末内のみに保管されます。クラウドに送信されたり、開発者に送信されることはありません。アカウント登録も不要です。

書類の取り込み

📸
取り込み画面（ファイル選択前）import_top.png

このアプリのワークフローの起点は PDFの取り込み。iサポ経由、クラウド経由、端末内ファイル経由など、複数の経路を用意しています。

このセクションの目次 共有ルート 「で開く」ルート 一括取り込み 取り込み後の修正 重複チェック

iサポから「共有」で取り込む（メインルート）

  1. iサポでPDFを開く

    公式アプリで図面・見積り・仕様書のPDFをタップして閲覧画面を開きます。

  2. 共有メニューを開く

    PDF閲覧中の画面から共有メニュー（OSの共有シート）を呼び出します。共有ボタンの位置や表記はご利用のOS・iサポのバージョンによって異なりますので、お使いの環境に合わせてください。

  3. 「ICHIJO施主手帳」を選択

    共有シートから本アプリを選びます。

  4. 自動取り込み完了

    アプリが起動（もしくは既に起動中ならフォアグラウンドに）し、取り込みが自動で始まります。図面番号・種別・改訂日付が自動検出されます。

「このアプリで開く」ルート

既定のPDFビューワを登録していない場合、iサポでPDFをタップした瞬間の「どのアプリで開きますか？」リストに本アプリが表示されます。これを選べばワンタップで取り込みが走ります。iOSでは「共有」の代わりに Safari・ファイルアプリからの「このアプリで開く」も同様に動作します。

多くの方は別のPDFビューワ（Adobe、Acrobat等）を既定に設定しているため、このルートが出ないケースも多いです。基本は「共有」ルートがメインと考えてください。

どちらの経路でも、取り込み時に端末内へPDFのコピーが作られます。元のPDFはそのまま残るので、誤って削除する心配はありません。

一括取り込み（クラウド・端末から）

Dropbox・iCloud・Google Driveなどに保管しているPDF、または端末にダウンロード済みのPDFを、複数まとめて取り込む方法です。

  1. 「PDF取り込み」タイルをタップ

    ホーム画面から。

  2. 「ファイルから選択」をタップ

    取り込み画面上部のボタン。

  3. 複数選択

    端末のファイルピッカーまたはクラウドから、PDFを複数選択できます。

  4. プレビューリストで確認

    選択したPDFが一覧表示され、各ファイルに自動検出された図面番号・種別・日付が表示されます。

  5. 必要なら個別編集

    検出が失敗した項目は、その場で手動修正できます。

  6. 「取り込み実行」をタップ

    一気にDBに登録されます。

取り込み後の修正

取り込み時に検出ミスがあった場合は、以下の項目を後から修正できます。

  • 図面番号
  • 種別（CAD / 電気 / 仕様書 / 見積り / その他）
  • 改訂日付
  • ファイル表示名

修正方法：ドキュメント管理 → 該当ファイル長押し → ファイル情報を編集。

重複チェック

同じPDFを2回取り込もうとすると、自動で重複検知されて「既に取り込み済み」の警告が出ます。バックアップから復元した直後など、同じPDFを再取り込みしようとしたときに便利です。

CAD図面・電気図面の2バージョンを重ね合わせて、変更箇所を色分け表示する機能です。

種別とバージョンの選択

📸
図面チェック選択画面（種別セグメント＋Dropdown＋履歴）diff_select.png

画面構成（縦方向）

  1. AppBar：「図面チェック」
  2. 種別セグメント：CAD図面 ／ 電気図面（SegmentedButton）
  3. 単独図面解析トグル（SwitchListTile）
  4. バージョンセレクター：旧／新の2つのDropdown（単独モード時は1つ）
  5. アクションボタン：「比較する」または「表示する」（単独モード時）
  6. 比較履歴リスト：過去の比較実績が新しい順に表示
  7. バナー広告（画面下部）

種別セグメントを切り替えると、対応するリビジョンだけが Dropdown の選択肢になります。

単独図面解析モード

差分比較ではなく、1つの図面を高画質で開いて 画面上にピンメモ したい場合のモードです。

  • SwitchListTileをONにすると、バージョン選択が1つに変わります
  • 「表示する」ボタンで該当ファイルを開きます
  • 画面構成は差分結果画面とほぼ同じですが、差分/新/旧の切替は無効

自動計算の流れ

  1. PDFを画像に変換（300dpi／グレースケール）
  2. 自動位置合わせ（3段階）：
    • グリッド検出でざっくり合わせる
    • ダウンサンプル画像で微調整
    • 原寸で最終調整
  3. ピクセル差分を計算（閾値30）
  4. 色分け：赤=削除／青=追加／薄グレー=変化なし

計算中は進捗バーと「準備中...」などのメッセージが表示されます。計算結果はキャッシュされるので、同じ版の組み合わせをもう一度開いたときは再計算なしで即表示されます。

ノイズ除去フィルタ

スキャンノイズや1pxの細かいズレで不要な差分が大量に出る場合、右上のドロップダウンから4段階のフィルタを選べます。

レベル効果推奨シーン
しっかり（デフォルト）フィルタなし、全差分を表示厳密に確認したい
ややざっくり1pxノイズ除去一般的な用途
ざっくり小さな塊を除去スキャンPDF
超ざっくり目立つ変更のみ表示概要把握

フィルタレベルは ページ別 に保存されます。1ページ目は「しっかり」、2ページ目は「ざっくり」のような使い分けが可能。レベル変更時は再計算が走ります。

調整済 / 元の差分 切替

手動で位置合わせを行ったページでは、画面右上に「調整済 / 元の差分」の切替スイッチが表示されます。調整結果を適用した差分と、自動位置合わせ時の差分を比較できます。

※フィルタレベルが「しっかり」以外の場合、このスイッチは無効になります（常に調整済を表示）。

比較履歴

差分選択画面の下部に、過去の比較結果が新しい順に並びます。履歴にはメモ件数・位置合わせ調整の有無・ノイズ除去レベルがすべて紐づいており、タップで瞬時に前回の状態から再表示できます。

  • 図面番号（旧 → 新）と種別（CAD／電気／単独図面）
  • 実行日時
  • メモバッジ（この比較に紐づくメモ件数）
  • 位置合わせ調整済マーク（手動調整を保存した比較に付く）

タップで比較結果を即表示（キャッシュ使用・手動調整値もそのまま復元）、長押しで「履歴を削除」の確認ダイアログが出ます。

位置合わせの調整値（オフセット量）は比較ID＋ページ単位で保存されるため、後日同じ組み合わせで開いても再調整は不要です。

操作方法

  • ドラッグ：旧図を上下左右に移動
  • ピンチ：全体の表示倍率を変える（基準線があるので位置は保持）
  • 長押し：基準点（アンカー）を追加
  • 右上「自動調整」：再度自動位置合わせを実行
  • 右上「保存」：現在の調整値を保存し、差分を再計算して結果画面へ戻る
  • 右上「リセット」：調整をクリア

項目ツリー

見積りPDFを取り込むと、以下の 8つの費用区分 に自動で分類され、折りたたみ可能なツリーで表示されます（①〜⑧の番号は一条工務店の見積書フォーマットに対応）。

  1. ①建物本体工事
  2. ②建築申請・その他業務
  3. ③付帯・屋外給排水等
  4. ④標準仕様外工事
  5. ⑤太陽光・蓄電池
  6. ⑥土地購入費用
  7. ⑦その他費用
  8. ⑧預り金精算費用

区分内にさらに小計行や特別割引（例：建物本体工事内の「法人契約特別割引」「ご紹介割引」等）があり、ネストして展開できます。

自動検算（整合性チェック）

取り込み時に、パースした各項目を合計して見積書の小計・合計と突き合わせる検算が走ります。ズレがある区分にはカード上に ⚠️ アイコンが付き、詳細画面でもズレ金額が赤字で併記されます。検算が通らない代表ケース：

  • 新フォーマットや法人契約の特別割引行のパース失敗
  • 手書き／スキャン取り込みのPDF
  • 一部項目の単価×数量の端数処理ズレ

⚠️ が出ていても、推移表や2版比較のソートには影響しません。

自動構造化（8セクション）

設備仕様書PDFを、以下の8セクションに自動分類します。これは一条工務店の「設備仕様書」PDFに含まれる表の見出しに対応しています。

  1. 外部仕上表：屋根・外壁・外構まわり
  2. 内部仕上表：床・壁・天井の仕上げ
  3. 設備仕上表：キッチン・浴室・給湯・衛生設備
  4. エアコン仕上表：エアコン・換気まわり（指示No付き）
  5. 収納仕上表：収納・クローゼット
  6. 内装仕上表：カーテン・クロス等の内装
  7. 設定外工事：標準外の特別工事
  8. 施工連絡表一覧：現場への指示事項

CSV取り込み

パワモニ側のCSVエクスポート

一条のパワーモニターアプリから、1時間単位の実績をCSVエクスポート：

  • ヘッダー：データ日時,発電量(kwh),消費量(kwh),売電量(kwh),買電量(kwh)
  • エンコーディング：BOM付きUTF-8または Shift-JIS（自動判定）

取り込み手順

  1. でんき通信簿画面を開く
  2. 右上のメニューから「CSV取り込み」
  3. ファイルピッカーでCSVを選択
  4. 取り込み進捗を確認

    ヘッダー検証→データパース→DB登録→月別キャッシュ生成 の順で進みます。

  • 既存データと重複：同じ日時のデータは上書き更新

計算ロジック（参考）

  • トータル収支 = 売電収入 + 太陽光自家消費節約 + 蓄電池節約 − 基本料金 − 買電コスト + 補助金
  • 太陽光自家消費節約 = 自家消費量(kWh) × その時刻の買電単価
  • 蓄電池節約 = 放電量(kWh) × その時刻の買電単価 − 充電時のオポチュニティコスト
  • 自給率 = (発電量 − 売電量) / 消費量 × 100%
  • 充放電効率 = 放電量 / 充電量 × 100%

バックアップ

項目動作
バックアップ作成PDFとデータベースをまとめてZipファイルに保存。共有メニューから保存先（クラウド・他端末等）を選択できます。破損検知の仕組み付き。
バックアップ復元Zipファイル選択 → 確認ダイアログ → 復元処理 → アプリ終了案内（iOSは手動終了）

復元の流れ（ステージング方式）

安全のため、復元は tmp領域への展開 → 整合性検証 → 最終コピー の3段階で行われます。検証に失敗した場合は、現在の端末内データには一切手を付けずに処理が中断されます。

  • 整合性OK：そのまま復元を実行し、完了後に再起動を案内
  • バックアップのバージョンが新しすぎる：旧バージョンのアプリで未来のバックアップは開けないため、アプリを最新版にアップデート後に再試行
  • ファイルが壊れている：転送中にZipが破損した可能性。バックアップ元からコピーし直してください
  • Zipの構造が壊れている：正しいバックアップファイルか確認

⚠️ 復元は既存データを完全に上書きします。 復元前に、念のため現在のデータをバックアップすることを推奨します。

困ったときは

PDFの取り込みができない・うまく認識されない

  • パスワード保護されたPDFには対応していません。保護を解除してから取り込んでください
  • 取り込みはできたが図面番号・種別が自動検出されない場合は、ドキュメント管理 → ファイル長押し → ファイル情報を編集 から手動で設定できます
  • どうしても取り込めないPDFに遭遇したら、お問い合わせ先からご連絡ください

図面の差分がズレている

  • 差分結果画面 AppBar の位置合わせアイコンから手動調整できます
  • スキャンノイズが多いPDFはノイズ除去フィルタを「ざっくり」以上にすると見やすくなります
  • 位置合わせの結果はページ単位で保存され、他のページには影響しません

見積りの検算でズレが出る

  • 新しいフォーマットの見積書ではパースが追い付いていない場合があります。設定 → 見積りデータ再構築 でパーサーを最新化してから再試行してください
  • それでも解消しない場合は、開発者モードを有効化して「パース失敗ダンプ」をお送りいただけると修正が進みます

でんき通信簿の数値が実際の電気代と合わない

  • 料金プラン設定を確認してください（特に燃料費調整、再エネ賦課金、基本料金、プランの適用期間）
  • プランの設定を変更した後は、右上メニューの 再計算 で全期間の再集計が必要です
  • 電力会社の検針日と月境界のズレによって、月別の合計に差異が出ることがあります

ウィジェットが更新されない

  • Android：バッテリー最適化から除外してください（設定 → アプリ → ICHIJO施主手帳 → バッテリー → 最適化しない）
  • iOS：WidgetKit が OS のタイミングで更新を行います。手動で最新化したい場合はアプリを一度起動してください

データが消えたように見える

  • まずデモモードが ON のままになっていないか確認してください（設定画面）。OFFに戻すと本番データに戻ります
  • バックアップZipがあれば 設定 → バックアップ復元 で戻せます
