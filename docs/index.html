📁

ドキュメント管理

PDFを取り込むだけで、図面番号・種別を自動検出。CAD・電気・配置図・立面図・見積り…すべての書類を図面番号ベースで一元管理します。

  • 複数PDFをまとめて取り込み、図面番号と種別を自動認識
  • 同じ図面番号のファイルをグループ化し、最新版を一目で把握
  • 同じPDFの二重取り込みを自動で防止
  • すべてのデータは端末内に保存 — クラウド不要で安心

図面チェック

「前回の図面から何が変わった？」をピクセル単位で可視化。CAD図面・電気図面の2バージョンを重ね合わせ、変更箇所を赤（削除）と青（追加）でハイライトします。

  • 図面のズレは「ピタッと合わせる」ボタンで自動補正
  • ピンチ操作で最大8倍ズーム、差分・新・旧をワンタップ切替
  • 比較履歴を保存し、過去の差分を即座に再表示
💰

見積り管理

見積りPDFを取り込むだけで全項目を自動抽出・構造化。手入力ゼロで金額の変動をバージョン間で追跡できます。

  • 8つの費用区分（建物本体・申請・付帯・オプション等）を自動パース＆検算
  • 2版比較で追加・削除・金額変更を項目単位で一覧表示
  • 推移表で複数バージョンの金額変動を横スクロールで俯瞰
  • CSVエクスポートで家族や営業担当との共有も簡単
📝

ウチメモ+

図面の閲覧中、差分チェック中、見積り確認中 — 気づいたことをその場でメモ。すべてのメモは元の画面・位置にリンクされ、ワンタップで戻れます。

  • 図面差分の上に直接ピン留め — 気になる箇所を座標付きで記録
  • 見積り項目に紐づけてメモ — 「この金額変更の理由を営業に確認」など
  • 全メモを横断検索、一覧画面から元の画面に即ジャンプ
  • CSV出力で打ち合わせ資料としてそのまま活用

🤖 Android

APKファイルを直接ダウンロードしてインストールします。Google Play ストアを経由しないため、端末の設定変更が必要です。

APKをダウンロード (v0.1.0)
インストール手順：
  1. 上のボタンからAPKファイルをダウンロード
  2. 端末の「設定」→「セキュリティ」→「提供元不明のアプリ」を許可
  3. ダウンロードしたAPKファイルをタップしてインストール
  4. インストール完了後、「提供元不明のアプリ」を無効に戻すことを推奨

プライバシーポリシー

最終更新日: 2026年3月23日

データの保存について

このアプリはすべてのデータをお使いの端末内にのみ保存します。外部サーバーへのデータ送信は一切行いません。

収集する情報

  • このアプリはユーザーの個人情報を収集しません
  • アプリに取り込まれたPDF・画像データは端末内のローカルストレージにのみ保存されます
  • バックアップファイル（Zip形式）はユーザーが明示的に操作した場合のみ作成・保存されます

第三者への提供

端末内に保存されたデータを第三者に提供・共有することはありません。
